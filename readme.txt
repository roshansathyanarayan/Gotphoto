2 python files are present inside images folder. 
sql-insert.py file is used to insert csv data into mysql Database. 
sql-read.py file is used to read data from mysql DB then perform transformations and write back a json Output with list of countries and count of people born there.

To start the docker, please run the below command.
1. docker-compose build
2. docker-compose up

To stop the docker please run the below command.
1. docker-compose down
